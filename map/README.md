 # Maps equality
Maps can't be compared using the == operator. The == can be only used to check if a map is nil.
``` 
package main

func main() {  
    map1 := map[string]int{
        "one": 1,
        "two": 2,
    }

    map2 := map1

    if map1 == map2 {
    }
}
```
invalid operation: map1 == map2 (map can only be compared to nil)
